# Fantasy Quest

A simple text-based adventure game in Python.

## Features
- Character creation with unique stats
- Multi-stage story progression
- Turn-based combat with random outcomes
- Inventory system with usable items
- Branching choices and multiple endings
- Error handling for invalid inputs
- Well-commented code for clarity

## How to Run
1. Make sure Python 3 is installed.
2. Run the game:
   ```bash
   python game.py
